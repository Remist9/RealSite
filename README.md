# 🛒 RealSite — Интернет-магазин

Полноценный веб-проект интернет-магазина, реализованный с использованием современного стека: **FastAPI + PostgreSQL + Tailwind CSS**.

---

## 🚀 Основной функционал

### 🔐 Авторизация и безопасность

* Реализована система регистрации и входа пользователей
* Используется **JWT (JSON Web Token)** для аутентификации
* Защищённые маршруты для работы с пользовательскими данными

---

### 🛍️ Каталог товаров

* Просмотр списка товаров
* Фильтрация по категориям
* Поиск товаров по запросу
* Удобная навигация по ассортименту

---

### 🧺 Корзина

* Добавление товаров в корзину
* Изменение количества товаров
* Подсчёт итоговой стоимости

---

### 👤 Профиль пользователя

* Просмотр личной информации
* История заказов
* Отображение будущих доставок

---

## 🧱 Технологический стек

### Backend

* FastAPI
* PostgreSQL
* SQLAlchemy (или используемый ORM)
* JWT для авторизации

### Frontend

* HTML / JavaScript
* Tailwind CSS

---

## ⚙️ Запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/USERNAME/REPO.git
cd RealSite
```

### 2. Запуск backend

```bash
cd src/server
python -m venv .venv
source .venv/bin/activate  # Linux / Raspberry Pi
# .venv\Scripts\activate   # Windows

pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

---

### 3. Запуск frontend

```bash
cd src/client
npm install
npm run dev
```
